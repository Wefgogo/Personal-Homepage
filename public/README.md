# Public Directory

This directory contains static assets that will be served at the root path.

## Files to Add

- `vite.svg` - Website favicon
- `resume.pdf` - Your resume PDF file for download
- Images and other static assets

## Usage

Files in this directory are served as-is. Reference them with absolute paths:

```html
<img src="/logo.png" alt="Logo">
<a href="/resume.pdf" download>Download Resume</a>
```
